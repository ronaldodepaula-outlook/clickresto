# Documentação Comercial – GitHub

## Visão Comercial
ClickResto é uma plataforma de gestão para restaurantes com foco em operação rápida, controle de pedidos e integração direta com API.

## Principais Módulos
- Cardápio (categorias, produtos, opções).
- Mesas e comandas.
- Pedidos e cozinha.
- Pagamentos e caixa.
- Clientes e endereços.
- Relatórios.

## Diferenciais
- Operação simples e focada em fluxo de restaurante.
- Integração REST pronta para mobile e painel administrativo.
- Separação clara entre UI e API.

## Público-Alvo
- Restaurantes, lanchonetes e bares.
- Redes com múltiplas unidades.
- Operações com atendimento no salão e delivery.

## Requisitos para Demonstração
- API ClickResto em execução.
- Banco de dados com dados básicos (mesas, produtos e categorias).
- Usuário com credenciais válidas.

## Estrutura do Repositório
- `saas/`: frontend administrativo.
- `clickresto-api/`: backend (separado).
